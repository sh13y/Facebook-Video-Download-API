How to Use

This page allows you to create direct links to the Facebook Video Downloader API without using a form.

Append these URL parameters to this page's URL:

  • url: The Facebook video URL (required)
  • quality: The video quality (optional, defaults to "best")
  • api: The API server URL (optional, defaults to "http://localhost:8000")

Example:

direct.html?url=https://www.facebook.com/share/v/1C6EwSc49J/&quality=720p&api=http://localhost:8000
